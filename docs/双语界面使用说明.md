# 双语界面使用说明

## ✅ 完成！保守方案已实施

应用现在支持完整的中文和哈萨克语双语界面，每个语言Tab都包含完整的功能。

## 界面结构

### 主界面
- **中文 Chinese** Tab：所有功能的中文版本
- **Қазақша Kazakh** Tab：所有功能的哈萨克语版本

### 每个语言Tab包含两个子Tab

#### 1. 语音合成 / Сөйлеу синтезі
完整的TTS语音合成功能：
- 输入文本
- 选择音色预设
- 创建/删除音色（快速操作）
- 启用LoRA
- 高级参数设置
- 临时音色克隆
- 开始合成

#### 2. 音色管理 / Дауыс басқару
完整的音色CRUD管理功能：

**音色列表 / Дауыс тізімі**
- 查看所有音色状态
- 对齐评分和状态
- 刷新状态

**创建音色 / Дауыс жасау**
- 输入音色名称
- 上传参考音频
- 输入参考文本
- 自动验证对齐

**编辑音色 / Дауысты өңдеу**
- 选择音色
- 播放音频
- 编辑参考文本
- 更新音频文件
- 保存修改

**查看详情 / Толық ақпарат**
- 查看音频信息
- 查看参考文本
- 查看元数据
- 播放音频

**删除音色 / Дауысты жою**
- 选择音色
- 确认删除
- 安全删除

## 使用方法

### 中文用户
1. 打开应用：http://localhost:7870
2. 点击 **"中文 Chinese"** Tab
3. 选择 **"语音合成"** 或 **"音色管理"** 子Tab
4. 所有界面和提示都是中文

### 哈萨克语用户
1. 打开应用：http://localhost:7870
2. 点击 **"Қазақша Kazakh"** Tab
3. 选择 **"Сөйлеу синтезі"** 或 **"Дауыс басқару"** 子Tab
4. 所有界面和提示都是哈萨克语

## 技术实现

### 翻译字典
所有文本都存储在 `TRANSLATIONS` 字典中：
```python
TRANSLATIONS = {
    "zh": {
        "tab_synthesis": "语音合成",
        "tab_voice_management": "音色管理",
        ...
    },
    "kk": {
        "tab_synthesis": "Сөйлеу синтезі",
        "tab_voice_management": "Дауыс басқару",
        ...
    }
}
```

### UI构建
- `build_ui_tab(lang)`: 构建完整的语言Tab
- `build_voice_management_ui(lang)`: 构建音色管理UI
- 所有函数都支持 `lang` 参数

### 函数支持
所有操作函数都支持语言参数：
- `create_new_voice(..., lang="zh")`
- `delete_voice_from_management(..., lang="kk")`
- `update_voice_audio(..., lang="zh")`

## 未来扩展

如果需要添加更多语言，只需：
1. 在 `TRANSLATIONS` 中添加新语言
2. 添加新的语言Tab
3. 调用 `build_ui_tab(new_lang)`

例如添加英文：
```python
TRANSLATIONS["en"] = {
    "tab_synthesis": "Speech Synthesis",
    "tab_voice_management": "Voice Management",
    ...
}

# 在主界面添加
with gr.Tab("English"):
    build_ui_tab("en")
```
